# Svelte Starter Project

## Project structure

```
.
├── README.md
├── index.html
├── package.json
├── src
│   ├── App.svelte
│   ├── app.css
│   ├── main.ts
│   └── vite-env.d.ts
├── svelte.config.js
├── tsconfig.json
├── tsconfig.node.json
└── vite.config.ts
```

## Clone starter

```bash
git clone --depth=1 https://github.com/secondary-smiles/svelte-starter.git svelte-starter
rm -rf !$/.git
```

## Init git repo

```bash
git init
git add -A
git commit -m "Initial commit"
```

## Setup svelte project

```bash
yarn
yarn dev  
```
